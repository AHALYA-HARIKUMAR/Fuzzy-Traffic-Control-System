# 🚦 Fuzzy Logic Smart Traffic Signal Control System

An intelligent smart-city traffic management system developed using Python and Fuzzy Logic to simulate adaptive traffic signal control at a real-world intersection.

---

## 📌 Project Overview

This project uses **Fuzzy Logic** to dynamically control traffic signal timings based on traffic density and waiting conditions instead of fixed timers. The system also includes pedestrian crossing management and emergency vehicle priority handling.

---

## ✨ Features

- 🚥 Adaptive traffic signal timing using Fuzzy Logic
- 🚑 Emergency vehicle priority system
- 🚶 Pedestrian crossing request system
- ⏱️ Real-time signal countdown timer
- 🧠 Intelligent phase-based traffic control
- 🏙️ Smart-city traffic intersection simulation
- 🎨 Interactive graphical user interface using Tkinter
- 🚨 Dedicated emergency lane simulation

---

## 🛠️ Technologies Used

- Python
- Tkinter
- NumPy
- Scikit-Fuzzy

---

## 🧠 Concepts Implemented

- Fuzzy Membership Functions
- Fuzzy Inference System
- Rule-Based Decision Making
- Smart Traffic Optimization
- AI-Based Signal Control

---

## 📷 System Simulation

The application simulates:
- North-South traffic phase
- East-West traffic phase
- Pedestrian crossing phase
- Emergency vehicle interruption phase

---

## 🎯 Objective

The objective of this project is to demonstrate how Artificial Intelligence and Fuzzy Logic can improve traffic efficiency, reduce congestion, prioritize emergency vehicles, and enhance pedestrian safety in smart-city environments.
